The Rules of HTML

Html is based on tags and attributes

<p></p> é uma tag
<a href="https://www.sapo.pt"></a> é uma tag a (anchor) com o attribute href com o valor https://www.sapo.pt

Html is not case sensitive

<p></p> é uma tag

Apesar de não ser case sensitive é boa prática termos todo o nosso código em lower case

Html has void and not void tags

<p></p> é uma tag not void, espera ter conteudo dentro de si. Logo tem uma opening e uma closing tag.
<img src="" /> é uma tag void. O seu propósitio está associado a attributes e não a conteudo interno.

Html is meant to be nested

Always close your tags

<p></p> é uma tag not void, espera ter conteudo dentro de si. Logo tem uma opening e uma closing tag.
<img src="" /> é uma tag void. O seu propósitio está associado a attributes e não a conteudo interno.

An HTML documents is divided in two sections, head and body

Things in the head tag are things that shouldn't be rendered: information about the page and how to process it. Things in the body tag are the things that should be displayed: the actual content. Javascript in the body is executed as it is read and as the page is rendered. Javascript in the head is interpreted before anything is rendered.
